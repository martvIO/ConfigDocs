Configure The Bot

We need to do some configuration before we can use the bot.

Setting up the Reddit app

First, you will need to get some credentials from Reddit. These values are:

REDDIT_CLIENT_ID
REDDIT_CLIENT_SECRET
REDDIT_USERNAME
REDDIT_PASSWORD

You can get these values from:
https://www.reddit.com/prefs/apps

Note that you need to be logged in to your Reddit account. This account cannot be a Google account or any other social login. It must be a Reddit account with a username and password.

Head over to the Reddit apps page. Scroll down and click the button saying:

create another app...

Fill in the fields:

Name: You can put any name, for example:
reddit-bot

Radio Buttons: Select the third option:
script

Description: Anything (not important)

About URL:
https://google.com

Redirect URI:
https://google.com

Great, you created the app! Now you need to give the values to the bot.

The text under "personal use script" is your:

REDDIT_CLIENT_ID

The text next to SECRET is your:

REDDIT_CLIENT_SECRET

DO NOT SHARE THIS WITH ANYONE.

The text where it says developers is your:

REDDIT_USERNAME

And:

REDDIT_PASSWORD

Is your Reddit account password.

Your information is not logged.

Obtaining TikTok SessionID

If you want to use the TikTok TTS, you need a TikTok account.

The process of obtaining a SessionID:

  1. Log in to the TikTok Web App:
    https://tiktok.com
  2. Install the Cookie Editor browser extension:
    https://cookie-editor.cgagnier.ca/
  3. Open the extension
  4. Look for:
    sessionid
  5. Copy it
  6. Paste it in your config.toml file:
    tiktok_sessionid
